一、晚上学习总结

1. python字符串

  • string.tetle() 单词首字母大写
  • string.lower() 所有单词小写
  • string.upper() 所有单词大写
  • string.strip() 清除空格
  • 注意这些都是临时的修改要确定修改必须进行赋值操作

2. Python字符串拼接

  • 使用 + 号进行字符串拼接

3. 使用制表符和换行符

  • \t 是制表符
  • \n 是换行符
  • 使用要求必须是在””中，因为这是字符串的操作

4. Python中3中引号的使用

  • ‘’ 或 “” 或 “”” “”” 使用是相同的但是后面多用于字符串文档，或进行排版
  • 在 “”中使用’’ （相反） 都没有问题，但是不能 在’’ 中再有一个’
